# c-language-project
#about:this project helps in comparision of  yeast growth rates by using different repurposed drugs.
it also tells which repurposed drug is most effective,moderately effective and not effective based on yeast population growth rate...
#example:
input:enter the number of drugs you wanted to test:3
enter drug name:aspirin
growth rate=3
output:moderately effective
enter drug name:disulfiram
growth rate=-2
output:most effective
enter drug name:paracetamol
growth rate=10
output:less effective
